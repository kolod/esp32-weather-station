Install CA Certificate

  1. Obtain ca.crt from whoever set up your weather station.
  2. Install it into your OS or browser trust store:
    • Windows — double-click → Install Certificate → Trusted Root Certification Authorities
    • macOS / iOS — open the file, trust it in Keychain Access / Settings → General → VPN & Device Management
    • Android — Settings → Security → Install a certificate → CA certificate
    • Firefox — Settings → Privacy & Security → Certificates → Import
  3. After installing, all weather stations signed by this CA will load without certificate warnings.
